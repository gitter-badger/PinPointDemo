The PinPoint Geocoder Project

My name is Tauren Kristich and I am a Software Developer living in New York. PinPoint is a side project I have been actively developing. It started out as a small Python module to parse out and cleanse addresses - particularly ones with bad or incorrect data.

The project eventually grew from an address parser to full scale geocoder - allthough, we're not there...yet!

The goal of PinPoint is to make a fully functioning and accurate open source geocoder built with Python, open data and PostgreSQL.

Curently, the geocoder supports only single addresses (aka no intersections). There is a lot of work to do so please feel free to help out!
